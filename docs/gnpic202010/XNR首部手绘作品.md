# 1

想画这些画的最初缘由是受到外界的刺激，我要证明给大家看不是某些自闭的网站就一定有好的作品，想看好作品也不一定要指望着哪些个隐居的画家。
刚好最近买了一个WACOM BAMBOO的手绘版，小试牛刀一下。

画这几张插画用了不少时间，请大家欣赏，有几张比较粗糙，请大家见谅，毕竟是我第一次在电脑上作画。
如果我们圈子内有足够的人才的话，完全可以制作我们华人区第一个GTS游戏，也让那些搞小圈子的同好羡慕一下。

我不收取金钱，因为在论坛里我不缺钱，我也不设定回复，因为我真心想跟大家交流想法，也想听听大家的意见和建议。
如果喜欢的人多的话，我再把原图发上来，如果不合你的胃口，也不要发牢骚，毕竟每个人的口味都不一样。
就说这么多，女神论坛已经倒了，我希望大家的热情不要倒，谢谢。。。

PS：这些图如果发到外国同好的网站上，会不会给这个论坛带来麻烦？暂时先不外传。

哈哈，我就是要达到这种抛砖引玉的效果！
其实说来我刚到但丁的论坛也是一直潜水的，然后开始做一些合成图，参与GTS文化的讨论。。。最后终于爆发啦！
说到作图的启发，其实还是《缩小学院》那个RPG游戏和那个叫SS的作者，因为我很喜欢他的作品的风格，关键是我觉得他的POV（第一人称视角）做的很好，很有感觉，所以也就产生了自己作图的想法，毕竟我也是有绘画天赋的人啊，哈哈。。。其实这些图的制作过程中也在参考他的作品呢。。。
说到制作游戏，我觉得只要有懂得游戏制作程序的人就完全可以，我会做插画，但丁可以写游戏的剧情，再找些人做一些其他工作，如背景音乐什么的，这样我们也能做一个很好的GTS游戏了。
让国外的同好在一边低调去吧，哈哈！
