# 1

鞋袜，穿在脚上，或保护足部，或使脚掌更加清新可爱，但对于漫步在城市中的巨大少女而言，鞋袜可能还会碍事。
那双无比巨大的脚掌，摆脱鞋袜后，贴在小人城市中，每走一步，都能感到舒适无比——好比在沙滩上无人干扰地踩着柔软细沙。……

-----------------------
久违，各位。
这时本人第二张手绘作品，构思加绘图共达4小时……

少女身高可达9km，脚掌大约1200m

因我的相机质量不敢恭维，所以全图可能很模糊，而缩略又看不到细节。所以请各位下载原图，而自己适当调节大小再看细节。
这幅画是2张a4纸拼合的。下张主要是少女大部分身体和少女脚下的情况，脚印、城市、大桥等等。上张是少女头部和飞过的飞机。
而最上边有两个小图，由线条框住。
那两张图是对少女足部的特写，细节较多，可以放大看看。
总之，本张图内容较多，但不会很粗糙。
