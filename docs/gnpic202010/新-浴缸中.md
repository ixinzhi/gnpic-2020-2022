# 1

少女正在浴缸中,她十分友好,允許小人們在她的浴缸中活動。許許多多的人類乘著不同的飛行器抵達她的肌膚和浴缸中。 

嘗試做這種類型的…細節太多了,用了我2小時…![](static/image/smiley/bigface/10.gif)  
在此說一下細節,真正還需你自己觀察.這些細節絕對是有的.
圖的左下,有一艘觸及少女小腿的游輪沉沒,幾艘救生艇開出.
圖的右下,有一艘遊輪和另一艘船隻,稍稍靠左有一架失事飛機墜入浴缸水中,一側機翼冒出濃煙.飛機放出了幾艘救生艇.
圖的中部,有一艘快艇靠近少女的***.
在圖中,有幾架直升飛機在運送小人到少女的肌膚上.少女正看著她眼旁的一架直升飛機.
 注意,水上的任何元素都有倒影.其中那架失事飛機有一部分墜入了水中,有折射.
 少女皮膚上的小人看似是個黑點,實際上放大到實際大小是一個較清晰的小人,決不是一個小點.

所以，請打開圖片後，先看一下縮略大小，再放大到原始大小看。

請回復,謝謝。
