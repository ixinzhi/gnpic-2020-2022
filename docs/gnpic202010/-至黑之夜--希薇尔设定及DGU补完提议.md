# 1

最近在和专业老师忙一个产品的造型设计，外加上白天有课，工作量骤增，关于【至黑之夜】的漫画化速度也降下来了

在之前两次的经验积累后，我想认真的，按部就班的把【至黑之夜】画出来，在看完站长的DGU系列设定以后，我心里想，哇，这也太棒了，漫画的世界观的框架、背景，但大都已经进本弄好了，这简直是省去了极大的准备工作量。大凡一部优秀的漫画一般都有一个完整的完善的世界观，比如FATE，比如EVA，所以我想在后续的时间里在站长建立起的这个世界观上完成【至黑之夜】这个漫画，不再是短篇，而是一个相对的中篇漫画，所以我亦没有急于落笔，而是开始着手先弄好各个主要人物的设定工作，后续还会有关于魔舰破滅之翼、主要城市草图等的设定。同时最近我又开始重温贞本义行老师的新世纪福音战士，这部漫画个人感觉无论是线条的风格还是分镜 布局谋篇都是GTS漫画非常值得借鉴的。站里的小说，《至黑之夜》和《极北圣歌》是我最喜欢的两部，因为这两部把GTS归入到了传统主流的小说中（当然还有好多优秀的小说，但是在刚刚接触GTS的我有限的时间里，主要接触到的是这两部），Sword同学在极北圣歌中提出了这个想法，我举双手赞同！而期望把GTS漫画也归入到能和主流漫画相接近的层次，是我最大的愿望，个人画技有限，我会不断尝试练习，调整画面的精度，这也会放慢出图的速度，但是这次，只要时间允许，我就会坚持画下去。

下面是关于但大DGU世界设定的一点提议，主要受最近EVA的世界观的影响

关于神女的起源：星之魂是否可以作为神女和人类的起源呢？星魂在不断进化的过程中，发生了自身的分裂，分裂的个体由于分裂时获取的能量不同，一部分大的个体成为了神女，另一部分小个体成为了人类，都在进化的过程中形成了人格。然而早期人类的先驱者发现了这段起源的历史，又发现了这些作为星之魂裂生的新机体的最终命运是重新归于星之魂本体之中（早期的希薇尔不断毁灭屠杀吞噬一个有一个世界即是这个论断的证明），那么能否战胜星魂就成为了人类与神女最终归于何方的问题，这些知识被写进一部叫做《先驱者的遗产》的信息载体，在先驱者的世界被希薇尔毁灭时，保存在一颗已死的恒星的核心中，等待有朝一日后人能够发现并寻求出新的出路……

下面是我关于这次讨论的总结：

1.希薇尔的设定：

A. 人类化以后成为比杰克矮一些的少女，恩但依然相对无口，是相对！和杰克站在一起大约是……大家想一下囧虚和长门吧，我大体是打算设定成这个身高和身材比例的。无口其实是个关子，希薇尔并不会一直无口化下去，而是一个男主角和女主角个人情感逐渐发展升华的过程，是从彼此不了解的隔阂到希薇尔对于杰克的精神上的依恋的过程，就好比碇真嗣和凌波丽之间的情感，一种在背负着十字架的人生中萍水相逢的依恋。在漫画中我打算将男主角的戏份再增加一些，作为女主角人生态度改变的根本原因，这样在杰克作出最后决定时，大家才会更深切的感觉到那种生离死别的残酷。而杰克最后的抉择也能够让希薇尔真正体会到在茫茫尘世间，还有一种能够令冰封千年的心再次悸动的情感。“死与新生”（PS：御姐和萝莉之间切换是个好点子耶！）

B. 关于男主角的设定问题：恩既然女主角的设定发生了改变，男主角的设定也就需要有所改变了。需要改变以后，我居然首先想到了阿虚……但是阿虚的外貌又和女主角不是很搭配了！这样我打算让男主角长一些胡子茬，这样的组合也许会更合适一些吧。杰克也许可以设定为一个在以往比较懒散，讨厌麻烦，没有耐心，不喜欢帮助别人的人，但是这种性格也在与女主角相遇后被对方改变了。就好像古龙说“往往并不是你自己去找麻烦，而是麻烦找上了你。”我自己曾想象也许杰克会用杉田智和的声音在冲向星魂时，嘴角泛起一丝微笑，对自己说：“呵，这还真不像是我的风格啊。”等设定图出来以后再继续探讨这个问题。

C. 世界观问题出现在我没有把站长的小说都读完上，这个先可以放一放。![](static/image/smiley/bigface/1.gif)

D. 背负着罪的人生，让男女主角旅途上的步伐总带着一份沉重，这个也是我想加入进去的，也就是给杰克弄一个令他自己有负罪感的过去，一个失误令周围人遭受了巨大的伤害，他当时选择了逃避。“杰克倚在角落，摸索着点起一支烟，喃喃自语：'我根本就是一个废柴么……'，随后就将头深深的埋下去。希薇尔静静的坐在旁边注视着杰克，红色的瞳孔中似乎有一丝情感闪过，嘴唇长了长，却终于拍拍裙摆，打开舱门离开了。”

争取在这两天搞出来人类状态的希薇尔和杰克的人设！
