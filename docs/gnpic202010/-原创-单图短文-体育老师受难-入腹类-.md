# 1

某日 教室内：“呼，呼。。。新来的体育老师真是恶趣味，一上来就拿长跑折磨我们。。。”“啊，啊，是啊，累死我了，到现在还喘不过来气。。。”“呼，呼，放心，班长已经去收拾她了。”“嘿嘿，是的，我刚才看见班长她飞到体育老师的水杯里去了。““啊，怪不得没看见她呢。太好了，有班长出面，可以杀杀她的威风！。‘与此同时，操场上：“呃。。。呃啊。肚子好疼。。。是吃坏东西了吗。。。““喂！臭女人！老娘我在你湿哒哒的肉袋子里呢！““啊？哎！你是谁？哪个班的赶紧出来！““还真不知天高地厚啊！你不需要记住我是谁，你只需要记住这个！“ （一拳打在胃壁上）（颤抖）“啊啊啊好痛啊，你。。。你是给我下毒了吗？可恶。。。诶哟！”“没错，老娘我现在对你最毒！新上任是吧，三把火是吧，上来就5千米长跑是吧？”（一脚踹在幽门旁）“啊。。。原来是这样。。。同学，老师知道错了，求求你饶了我吧，老师以后绝对不这么搞了。。。”“谅你也不敢。以后不折磨我们就好说！但是今天这顿打你的胃是挨定了！中午饭不用吃了,让你感受下跑步跑到胃痛的感受！”“同学。。。你若现在就出来，我保证取消所有5千米长跑。。。”“哼，你这肉袋又热又潮，我还不稀罕呆着呢，不过要是你出尔反尔，我就钻到你肠子里，让你当众出大丑！”“同意。。。同意”“行，把你嘴张大点，我可不想撞到你牙齿上。”“呜呜，好的。。。不过。。。3千米可以保留吧。。。”“啊？你说啥？” （用力撑大贲门）“啊啊啊。。。只是开玩笑。”就在一瞬间，班长顺着一道金光从老师口内穿出，回到了教室。
