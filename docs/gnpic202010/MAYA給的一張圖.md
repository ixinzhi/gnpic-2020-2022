# 1

念念碎一句：這個世界上的MAYA的模型換真是少的可憐.......

3DMAX不會用![](static/image/smiley/bigface/13.gif) .....操作起來和MAYA正好相反，好多快捷鍵都快把我逼瘋了.......本來想直接在3D裡面渲染出來了事的，但是操作實在是....... 於是想把3D的一個建築的超高精細度的模型導入的MAYA————在轉化過程中內存瞬間被吃的死機了（我承認我的機器不夠格......）

![](static/image/smiley/bigface/27.gif) 所以只能很偷懶的只把LOLI的模型導入MAYA了......  然後，又發現貼圖丟失.....材質亂套......高光亮的嚇人.....沒有背景....... ![](static/image/smiley/bigface/35.gif)

從找模型到渲染出結果，斷斷續續的改了一個晚上.............

總之，這是作為我學MAYA以來的第一份禮物，之後我也會更加努力，慢慢的從現在的呆板的男性大眾臉模型進化到口耐的LOLI模型。2D控不要失望，MAYA裡面是有卡通渲染這一說的，所以渲染出來的結果和平時看的GAL遊戲CG相差不大，這個模型的作者能渲染出超華麗的仿2D的圖，但是無奈我技藝不高............
