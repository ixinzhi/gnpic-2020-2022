# 1

最近看到论坛中勇于尝试贡献作品的人像雨后春笋般比例巴拉的出现 这让我隐居多年的老一辈情何以堪 所以我准备在情人节贡献一份自己的原创作品（只是画的时候赶上情人节/抠鼻）

论坛中我最喜欢的角色就是兔子了（/////） 所以我让她做主角 - -

我解说一下

兔子吞了个小人 小人具有超能力 历经千难万险终于从兔子口中逃出来了

我认为这个作品很适合改编成人气漫画或者做个5,6百集的tv动画
