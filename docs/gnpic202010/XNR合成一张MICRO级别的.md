# XNR合成一张MICRO级别的

作者：xnr

TID：2224

 

# 1

挖到一张吴佩慈的美图，忍不住做了一张。。。

看点在于这张图片非常非常大，大家可以找找有多少小人。。。

我就是很喜欢这种超大的图片，这样才会觉得自己很渺小。。。才会有GTS的那种震撼的感觉。。。

学校的电脑里的真人图的素材真的是非常的多，而且质量都非常的高。。。但是现在没有时间做了。。。只能忙里偷闲了。。。

所以嘛，我的原则是图片不够大的不做，GTS不够漂亮的不做，没有互动性的不做。。。

这样我可以节省时间，也不会给大家带来粗制滥造的作品。。。

另外抱怨下。。。论坛上传附件的最大容量是多少啊。。。这张图片做出来可是有8.67MB啊。。。

原图压缩为RAR文件也有6MB多啊。。。

压缩了一半4MB多居然还传不上去。。。

3MB多居然也传不上去。。。完了完了，你们损失大了。。。giantesscity最大只能传5MB，他们也损失了。。。

2MB多也传不上去。。。呜呜呜。。。那这张图还有什么意义。。。你们连小人都看不到。。。白花了我那么多时间。。。

终于传上去了。。。不知道能不能修改论坛的限制。。。我记得原来CITY那边是没有上限的。。。
