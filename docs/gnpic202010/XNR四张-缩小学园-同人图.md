# 1

画这四张花了不少时间。。。经常熬夜到很晚。。。**总共只有四张图**
每张图的尺寸格式我都是参照RPG VX的格式制作的，完全可以直接应用于游戏当中。

在每张图的下方，我都通过在线翻译网站写了一段日文，内容如下：

感谢《缩小学园》制作组创造的伟大作品！
在此用我的同人图向你们致敬！
——来自中国的GTS FANS：XNR

写这段话的目的是想发到日本和美国的GTS网站，但是又想避免引发道义和版权的问题，才出此下策。
请原谅我的虚荣，我很想把我的作品与海外的同好，尤其是日本的同好分享，也希望能在他们的圈子中积累一点人气。

当然，**首先我要在这里请大家确认，我这种做法没有问题，**我才会去发到日本的 GTS/Shrinker uploader 和美国的giantesscity。
如果可以的话，我还希望论坛里懂日语的同好能帮我修改下文字的内容，在此先谢谢了！

**下面是同人图的介绍，最后有提供无文字遮挡版的原图，只要参与回复和讨论，都可以看到。
在下希望大家能热情的支持和回复，更希望能抛砖引玉，促进大家的讨论和参与度，谢谢！**

我先说明下，明天我就要去外地工作了，时间真的非常紧迫，我是今晚赶工才画完的，所以有很多不尽如人意的地方我非常抱歉！毕竟本人的绘画水平、能力、经验和时间都非常有限，本人也只是业余绘画爱好者而已，不敢与专业画家相提并论。
我就回应几点吧：
1.人物的动作、面部表情等问题是我的水平的问题，没办法与日本的同人图画家相比，不过我以后会慢慢改进的。
2.POV视角一直都是很难以把握的，从透视角度看，都是上面比较小，下面比较大，我是尽量描绘出更真实的感觉，但是无奈经验很有限，所以不敢与白兔和SS相比。
至于你说的腿粗等问题，我也知道，不过目前也就只能这样了。。。其他细节问题说到底还是时间和精力的问题，我经常偷懒。。。跟SS看齐就好，哈哈。。。
3.最后那张图的建筑，我是参照SS那张图做的，他那里的建筑也都是一个样子。。。他也偷懒了。。。你说的踩踏建筑的冲击，其实做起来是很简单的，用的是PS里的smudge（涂抹污点）工具。
