# 1

（前言）用了两张张嘉倪的图，总体说图像变化不大，大多数还是文字了，勉强可以联系起来。
总序：在与我们平行的一个次元里，有着一个与我们世界相差无几的星球——海蓝星，在哪个星球上也生活着人类，只不过世上不存在两种一样的物体。所以哪个星球上的人类与我们的身体虽像，但他们却走上了和我们不同的路。在我们的世界，机械文明是我们立足的根本，而在那个世界，人们认为自身的修炼才安身立命的根本，所以，一种叫魔法的科学诞生了。在经过动荡的几千年后，在海蓝星，魔法终于粉碎了机械，人们也终于确定了他们的文明体系——魔动文明。在这个文明里，人人都会魔法，魔法的强弱是评价一个人的标准。伴随着这个标准，人类成立了最高魔法议会，有十二名大魔法师组成，他们代表了人类的最高权力，负责管理魔动文明的发展与运行，可是生物是有欲望的，人类作为高等生物在欲望方面尤为体现，于是乎，在议会成立的一年后，随着各个大魔法师的私欲与斗争，被迫解散了。于是乎，每个大魔法师都聚集里一群信仰自己的人开始了战争。史称第一次魔法大战，在战争中，人类损失了6名大魔法师和无数的人口，在战争后人类吸取教训成立了各个魔法联盟，也就相当与我们世界的各个国家。各个国家的首领就是昔日的那6名大魔法师，人类就相安无事的几百年。
几百年之后，随着大魔法师日渐孤寂的心灵和对于彼此的憎恨，新一轮的大战就此展开，可是战争刚一开始就被迫结束了。结束的原因是随着魔动文明的发展，人们开始追求于神的领域，人们不断研究禁术，妄想成为神。终于在开战的第30天后，随着一声巨响，塔拉奥联盟的大魔法师辛西娅最年轻的弟子也是最不待见的弟子——仅有21岁的欣欣.安妮成功的将禁术融入自身。在她成功后的第一件事就是将自己的老师辛西娅和她的一班弟子全部吞食，通过吞食这么一群法师她成功的稳定了禁术的能量，这份能量使她成为了魔动文明的第一位女神——魔法女神，简称魔女。她成神的当天，她以传播禁术的理由邀请了魔动文明最顶尖的20万个法师，也是这次战争的主力。每个法师都无法拒绝这个诱惑，于是他们在明知有险的情况下还是赴邀而去。当他们到达之后，他们才发现自己的魔力在这个魔女面前如同蝼蚁一般，即使他们将魔力集中在一起，也是如此，但更让人无法接受的是，他们的生命似乎对于魔女来说，也是如此…………………………
