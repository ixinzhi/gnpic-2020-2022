# 一笑

作者：fllg

TID：19718

 

# 1

天下渺渺，尽握掌心。

明媚皓齿，俯仰众生。

吐气如兰，风云变色。

曼妙舞姿，毁天灭地。

四方惊惧，屈膝来拜。

回眸一笑，尽收美腹。

堂堂人类，可悲可笑。

称霸万年，从无敌手。

可怜今日，灭于一笑。
