# 1

前些天，突发奇想，想要画个系列故事。
萌发了绘制个《谜之国的小动物饲养调查》。
基本上算是首次比较正式的画带故事性的漫画，感觉画故事漫画真心不容易。
由于时间精力有限，而想法却又极为庞大，抑制不住的希望尽快推进故事的发展，还想尽早与大家分享，就这么潦草的放上来了。还请大伙谅解哈。
至于有什么建议想法的，也可以畅说欲言哦，比如为角色取名，提供故事情节发展等
后续的剧情我将不定期持续更新。。。
![](static/image/smiley/bigface/54.gif)
8月12日更新3张
现在又觉得小动物饲养调查这个剧情不大好。。。![](static/image/smiley/bigface/57.gif)
之前头脑发热的想了这么一出，创作中发现并不理想，所有的剧情一直只有有个模糊的概念，现在的状态就是边画边编，再加上我这一拍脑袋一个想法的跳跃思维，对剧情还是不要有什么期待了。
