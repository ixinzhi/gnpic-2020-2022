　　姍躺在浴缸中，任憑燒開的洗澡水漾出浴池。
　　凌晨一點。
　　家中只有她一人。
　　父母都出去了，因為母親最近參加什麼娛樂節目，獲得了一百萬的獎金。多出來的一百萬讓兩個同時擁有婚外戀的夫妻從此分道揚鑣，姍不知道他們是如何分配的這筆錢，但是她知道，他們已經一個多星期沒有回家了，而且以後也不會回來。
　　任何電話號碼都找不到他們。
　　自己和這個每個月底都會拖欠租金的房子，還有幾件不值錢的家具，永遠被拋棄了。
　　不過現在，她根本不會對此感到什麼怨恨。
　　因為被學長強姦遠比這要痛苦百倍。
　　姍在放學的時候，終於鼓勵自己，向著憧憬已久的學長小田告白了。小田立刻就答應了，他們一起享用了晚餐，在包房中唱了幾個小時的ＫＴＶ，看了那場剛剛上映的[３Ｄ科幻電影](http://www.56.com/u91/v_NDg0ODQ1NjA.html)，在入夜之後，小田帶著她來到了一個神秘的地方，據說是小田的秘密基地。
　　那是一座爛尾樓，據說開發商欠了很多錢，建到一半就放棄了，好多年都沒有人再問津。
　　小田轉過身，充滿愛意的對姍說：“我們在這裡做吧。”
　　“什麼？”姍羞愧的退後了一步，“我們才剛剛在一起，不是應該一起學習，逛街，最後才發展到去對方的家中……”
　　“什麼啊，現在還有這麼保守的女孩麼？”小田已經抓住了姍的手腕。
　　“別這樣，小田…啊！！”
　　“這裡不會有人來的，最多是一群癮君子，他們對毒品更感興趣啦，不會打擾到我們的。所以，你盡情喊吧，我也喜歡姍你喊的聲音更大一些。 ”
　　“不要，小田，住手！！！　我求求你………”
　　……………………
　　姍再也沒有勇氣活下去。
　　被父母拋棄，沒有錢生存，被小田強姦………　死掉算了吧。
　　姍從樓上跳了下去。

　　**　第二章　蠶　上半部**　

　　姍走出浴室，擦拭著身體走回自己房間。
　　今天發生的一切似乎是夢，黑暗中，姍輕輕撫摸著自己的胴體。
　　應該被折斷的手臂、已經被小田咬破的乳房、被劃破的小腿、都完好無損，好像從一開始就沒有發生過一樣。
　　姍的視線突然落在了月光下閃閃發光的長發。
　　修長又順滑的長發，是姍一直以來想要的，在自己快要死的時候，是那個女孩救了自己。
　　她像個天使一樣發出光芒，當光照耀在自己身上時；
　　自己的衣服和傷口完全復原了。
　　姍跪在她面前。
　　天使把她的幾根長發放到姍手中：“你已經死了，但這幾根頭髮可以暫時延長你的性命，只要把一根頭髮放在身體或者別的什麼附近，它 們可以實現你一些小小的願望。不過，等到頭髮用盡，你就會真正的死去，我們不能過多的干涉這個社會，也不能改變你死亡的事實，抱歉， 這就是我能給你的一切了。如果你說出其中的秘密，或者被外人發現了你所許的願望，你和你的願望，都會泯滅。 ”
　　姍突然覺著這一切幼稚的很好笑，卻又無法對此產生懷疑。
　　她爬在床上，四肢順著自然下垂的角度打開，像一直受傷的飛蛾。

　　翌日

　　“早啊，姍，雖然有些晚，但是今天早晨，的確忘了給你打招呼呢。”小田轉過身看著身後的姍，他的笑容溫柔的近似於殘酷，早在一年前，他的笑容就已經把姍俘虜了。 “把我叫到天台，是為了享受晨風麼？的確，這很舒服呢。”小田說著敞開雙臂“姍，你要不要也試一試呢？ ”
　　“小田！”姍再也無法再平靜的吼出聲音。
　　“不要喊啊，我們距離很近，我聽得很清楚。”
　　“昨天，為什麼要做那麼過分的事情。”回憶讓姍下意識的把雙手擋在胸口。
　　“抱歉，我咬的太狠的，很疼嗎？”小田關心似地伸過手。 “抱歉，我當時太激動了。”
　　“不要！”姍一把打開小田的手，哽咽著說“小田！你為何要…… 我們才剛交往啊？”
　　“和交往多久沒有什麼關係吧？我們都說了，相互喜歡對方啊，而且我也一定會對你負責的，結算現在要結婚我也會同意的。然後，你就做一個全職的太太，賺錢是男人的工作。 ”小田滿面笑容的發表的自己的大男子主義的演講，並一步步的走進姍。
　　姍的步伐僵硬的後退，瘦弱的臂膀已靠在了天台的護欄上。
　　“住手————————”
　　一個長發竄入了小田的身體。
　　他的身子突然縮小了，自己衣服反而把他埋在下面。
　　他從衣服堆裡面爬出來，看見了一雙和他身高幾乎一樣的圓頭的黑皮鞋，順著包裹著小腿的白筒襪一直看到短裙下面的粉紅色內褲，再往上就是姍低垂著的臉。
　　頸部一陣酸痛。
　　“天，這是怎麼了？”小田恐懼的看著四周的一切，他現在只有幾厘米高。
　　姍稍微移動了一下自己的腳。
　　小田聽得見粘在姍鞋底的沙粒與地面摩擦的聲音。
　　一陣恐懼裹住了這個男人，只要姍抬起腳踩在自己身上，再像剛才那樣稍微擰一下，自己會被變成姍腳底下的肉醬了。

　　姍的呼吸突然急促起來，一想到真的要殺死小田，就連害怕起來。
　　真的要殺死他嗎？
　　自己真的這麼憎恨小田嗎？
　　不，雖然小田很過分… 自己還是可以原諒小田的，在這個世界上，喜歡自己，接受自己的人，也只剩下腳下這個只有幾厘米的男人了。
　　如果連山田也死掉…
　　姍無法自己想下去。
　　現在想想，昨天晚上也並不都是痛苦的回憶。
　　不過，自己一定要改掉小田那些總是自以為是的毛病。
　　這次，就當是對小田的一次處罰吧。

　　姍微微一笑：“小田，我可是一個魔女哦，現在你被我縮小了，所以一切都要聽我的了。”姍彎下腰，把剛剛想要逃跑的小田輕輕握在手中。
　　小田在自己掌心掙扎，好像剛剛離開水的魚，姍不斷提醒自己要輕一些，不然真的會傷到小田的。
　　姍平開手，讓小田能站在自己手掌上。
　　“好吧，你倒底想做什麼…… 我們的世界，怎麼會有魔女？”小田在顫抖著，這一點微妙的震動傳遞到姍的神經上，讓姍感到莫名的興奮。
　　“如果，你想知道的話，我可以慢慢說給你聽，不過，你要保證，要聽我的話。”姍把小田放到自己面前。
　　小田看著姍的巨大眼球，裡面的自己正一絲不掛的顫抖，他只感覺到一陣羞辱，自己從來沒有這麼難看過，這不公平，世界上怎麼會有魔女這樣的東西。
　　“我聽。”小田想像不出什麼對策，只有先答應對方了，更何況，看上去姍並不想要自己的性命。
　　“閉上眼睛。”姍用半命令的口吻說。
　　他要幹什麼，不會要把我的身高變回來吧？小田心中想著閉上雙眼。
　　姍的清唇輕輕親吻著小田，小田感覺自己全身都突然軟軟的，觸電一樣。
　　“呵呵”姍露出可愛的笑容“這只是第一步，學長昨天晚上對我做的事情，要用一生來補償。”

　　“喀拉”天台的門被推開了，班上的一對情侶葉和震有說有笑的走了上來。
　　“那不是小姍麼？”小葉朝著自己打招呼。
　　糟了，不能讓別人知道這件事情，不然學長和自己都會死的。
　　可是把學長藏在哪裡呢？
　　女生的夏季校服是沒有口袋的。
　　姍慌忙中張嘴把小田含在口中。
　　小田感覺四周一片漆黑，巨大的舌頭不斷的蠕動著，濕熱的口腔立刻就把自己包裹了起來，小田不斷把擠壓自己的舌頭踢開。還好姍的雙唇之間留了一道縫沒有閉合，自己得以呼吸新鮮的空氣。
　　姍的感覺比小田好不了哪去，含著一個活人比含著一塊糖難受多了，小田不斷的踢著自己的舌根，這讓姍需要不斷忍住嘔吐和吞嚥的感覺，要保持一道縫並不容易，因為現在自己的口水分泌不斷加劇，在這樣下去，口水都快要溢出來了。
　　“嗚嗚……”姍無法說話，只能發出嗚嗚的聲音。
　　“怎麼了，姍？這些衣服是？”小葉的眼光落在小田散落的衣服上。
　　糟了，再不解釋會被誤認為是偷走男生衣服的變態的。
　“我， 沒……”姍努力的張開嘴想說話，剛剛開口，卻聽見食道發出咕嚕一聲，小田被姍一不小心活吞了。
　　姍的面色突然變的煞白。
　“到底怎麼了嗎？姍，你怎麼不說話？”小葉關係的問道。
　“我，我。”姍急得快要哭了出來，小田會被自己消化掉的，得快點把小田吐出來！ “我去廁所。”姍飛快的跑下樓去。

　　被從狹窄的食道吞了下去，小田感覺自己都要被擠扁了。他想呼喊，但是胃裡面的空氣讓小田無法呼吸。食物消化發出的氣體、胃液、和吞嚥下的空氣混合在一起，發出嗆人的氣味。姍一跑動，半消化的食物就劈頭蓋臉的朝著小田臉上打過來，小山整個人像一個被丟入洗衣機裡面的衣服。

　　姍在洗手間外，拼命的用手指扣著自己的喉嚨，胃一陣抽搐，終於又一些東西被吐出來，但是食道口突然被卡住了，嚥下去的時候就很勉強，現在要吐出來哪有這麼簡單？姍能感覺到，小田在胃部通往食道的地方拼命的掙扎，要不了多久，小田就會一命嗚呼的。
　　“不要…怎麼辦？”姍焦急的說：“小田，你能聽到嗎？快出來。”
　　小田可以聽到，在姍的胃中，反而聽的很清楚，但是自己缺無法開口。
　　對了，姍突然想到了頭髮，姍又抽出一根長發，對它說：“求求你，救救小田，請保護他。”
　　頭髮消失了。

　　小田接近絕望了，難道自己會死在一個女生的胃中麼？
　　他躺在胃液中，嗆人的味道讓他無法睜眼，不過現在，即使睜開眼也看不見什麼。
　　他屏住了呼吸。
　　半分鐘過去了。
　　但自己絲毫沒有半點胸悶的感覺，反而，他感覺這裡的味道也沒有這麼嗆人了。
　　怎麼了？姍做了什麼嗎？
　　“學長，你在嗎？”他聽得見姍的聲音。
　　“在，你聽得見嗎？”小田大喊。
　　“聽得見，很清楚。”姍感覺小田發出的聲音好像肚子餓時的聲音。冷靜下來之後，姍突然想笑。為何要許這個願望呢？直接許願，讓小田直接出來不就好了嗎？姍準備拿出另外的頭髮：還剩四根。
　　姍猶豫了，她想起了那晚天使告訴她的，這些頭髮可以用來延續自己的生命。
　　姍從裡面拿出一根頭髮，心想著，這根頭髮是把小田復原的，於是單獨放了起來。
　　這樣就剩下三根。
　　能延續多長時間的生命呢？
　　一年？十年？二十年？
　　還是一個星期？或者幾十個小時？
　　當自殺的人不再想死，死亡的恐懼會變本加厲的襲來。
　　“姍！姍！”小田的呼喊聲打斷了姍的思考。 “讓我出去啊。你是魔女吧？”
　　“我當然是了，現在你不會被消化，也不會被悶死；不過，本魔女的魔力也是有限的，我沒有額外的辦法讓你出去了。”姍裝出一副了不起的口吻。
　　‘額外的？ '小田思考著這句話，也就是說，現在只能用一種方法了？食物排出體外的唯一的方法……“不要啊。”小田的大喊。
　　“這是對你的懲罰哦，學長，是對你昨天晚上對我做的事情。”
　　“和喜歡的人做愛有什麼不對的！你不是也喜歡我麼？”
　　姍羞愧的說：“不對，不對，不對，看來學長吸取的教訓還遠遠不夠呢。”
　　“啊！啊！我錯了，請原諒我把，放我出去。”小田知道，好漢不吃眼前虧，更何況，和便便一起排出體外，這太令人噁心了。
　　“即使這樣，我也沒有額外的方法了，所以，就委屈你了，學長~”
　　重新走回天台的路上，幾個女生望著姍發出一聲嬉笑。姍這才注意到，在外人看來，自己一直像個傻瓜一樣自言自語。
　　不過，即使被當做傻瓜，姍仍然會心的笑了起來。
