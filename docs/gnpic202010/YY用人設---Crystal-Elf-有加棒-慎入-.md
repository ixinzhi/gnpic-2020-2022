# YY用人設 - Crystal Elf（有加棒，慎入）

作者：gtsferygood

TID：5562

 

# 1

最近功課比較忙，所以很少浮上

為了表達歉意，所以就弄了個可以給大家YY的人物設定出來

這人設沒有正式的名字，所以就弄了個外號叫「Crystal Elf」

中文翻過來有「水晶般的精靈或妖精」等意涵

在此先感謝寺田落子 (ochiko) さん於昨晚在繪圖上的協助

Crystal Elf是跟史萊姆是完全搭不上邊的生物，

正確而言是一種元素體，外表呈現半透明固體狀

體色帶了點青藍，也有生長與人類相同形狀的全透明器官

但這些器官沒有實質上的消化與生殖作用，純粹是為了獲取快感而生

智商高低因成長過程而異，普遍來說是與人類差不多的

平時大小約三公尺的mini級左右，一但吸收了大量的水元素

最大可以增長到giga級的程度，而且也能自由長出輔助用的類陰莖組織

體質非常敏感，只要一有東西進入身體就會開始分泌大量液體

但是她們沒有痛覺，皮膚也相當的有彈性

所以即使是大型建築物也能輕鬆塞入她們的類陰道組織

而且能從外觀上清楚看見內部發生的一切

她們不需要進食生物，只要吸收天地精氣就能存活

因此小人被吞進去也是會伴隨著黏稠液體，而被完整的排泄出體外

由於體內也是存在著每日增生的水元素，為了排放累積過多的部分

會以排尿或搾乳的形式放出水柱，達到解除負載的效果

出沒的地點大多在高山與深林，總數量目前尚未估計出完整的數值

最大的弱點是無法耐熱，因此不會出現在沙漠、荒漠或火山地帶

她們的好奇心甚強，對活動的人事物都有高度的興趣，

所以也有Crystal Elf追著船隻而來到都市的情形

帶來的損傷不亞於一般的女巨人或其他物種
