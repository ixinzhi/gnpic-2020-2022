# 1

这个花了我不少时间，用SAI+PHOTOSHOP，手绘+合成。背景不是我画的，其他都是我画的。。。
图片很大，可以打开来看。。。
我画这幅画的时候才深刻感觉到，要达到白兔或者SS的水平并不容易。。。但我希望我的每幅新作都会比上一次有所进步。。。
今后恐怕没有什么时间作画了。。。毕竟我不是专业的画家。。。但我希望未来能为我们的游戏制作插画。。。
