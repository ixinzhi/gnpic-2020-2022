# 1

**第二章**       故事大體講的是：惠美第二天帶著哥哥小希出去去散步，順便會見一同考入東京大學的高中同學**千尋**，小希對千尋一見鍾情，冒著生命危險逃離惠美的「魔掌」，奔向千尋的「懷抱」。
       這次**沒有加劇情**，一是加劇情太累，很希望**GN劇情組**能幫這個忙。再則沒有劇情也無所謂，因為這樣可以**讓讀者盡情發揮自己的想像力**，從而更好地互動，呵呵。
        這次3D製作採用目前世界比較流行的**"高動態圖片照明(HDRI)"**技術，希望大家能夠喜歡。

        **誰想添加劇情就直接寫這個帖子裡吧，大A會感激你的！！**![](static/image/smiley/bigface/64.gif)
