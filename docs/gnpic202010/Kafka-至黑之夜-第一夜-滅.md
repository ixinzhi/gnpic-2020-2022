# 1

首先向大家道個歉，因為各種各樣的原因，沒有兌現及時畫出【至黑之夜】第一話![](static/image/smiley/bigface/57.gif) 最近先是感冒了，然後電腦有完蛋了等等……總之畫的時間一直很少，這幾天終於安定下來，才有功夫把第一話的草稿整理出來。製作中我主要參考了貞本義行老師的《EVA》中的分鏡方法，文字部份儘量使用站長的原文，保證和原著的貼近程度。畫風上我還在不斷的嘗試，希望自己能通過不斷地聯繫，將風格和線條逐漸成熟起來！
    我將至黑之夜的全文大約整理成了25話，也就是扉頁上第一夜 滅 到第二十五夜 死與新生，每一頁預計大約在10頁上下吧。現在草稿已經畫到了第3夜，過程中體會到不親身經歷很多事情就不會瞭解到裏面的困難，所以畫完第一話，再上網看漫畫時，常常會體會到一頁話背後畫師的努力了，賣油翁說“我亦無他，唯手熟爾”，看來面前的路還有好長好長要走。
    如果大家發現某人長得和碇XX很像，嗯，全當是巧合哈~是巧合哈~如果型月廚們發現有一段話的結構很……恩那是致敬哈！致敬耶！因為我太喜歡那段話了！
    恩不在碎碎念了，下麵是第一話，還是那句老話，請大家多多指教，大家多提意見，Kafka才能進步更快！
    恩再打個預告……第二夜 夢的起始音 大約2周后出圖，還請大家多多支持！<title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

恩第一次读至黑之夜的时候我就想到这个关于鞋子搭配的问题啦
不过在参考了一定量的哥特服饰风格后我得出鞋子能体现出精致纤细的特征的话，也不一定必须用皮鞋这个想法，比如我在任务设定时加入了很多细皮带束缚的装饰元素，为的是和凉鞋的细带协调，嘿嘿。个人揣度一开始站长设定穿皮鞋也有一层因素在于原作这个桥段中的希薇尔是小女孩，穿洋装的小女孩还是适合穿矮跟的童鞋的，这样看上去更萝莉吧~如果让小女孩穿上这种比较性感的高跟凉鞋，确实很不合适耶。
这个是反派女初步设定，不是很满意，因为原本希望她的服饰风格至少也要和希薇尔的服饰风格较为接近些……名字还没太想好，随便先取了一个![](static/image/smiley/bigface/57.gif)
至于她的身份，初步设想是星魂分裂出的另一个个体，对于希薇尔的感情很密切，称其为“姊姊”，当然希薇尔不知道这个半路跑出来的妹妹是从哪里钻出来的，至于这个妹妹是星魂和哪个亡者融合出来的，这个亡者又是何种身份背景，还没编出来。大体是这样的一个构想，提交给但大。这个设想主要源自《Blood+》和《Canaan》中的2个女主角的关系。露西亚的最终目的是想和希薇尔融为一体，重新成为完整的星魂，最终章也许可以添加融合为一体的希薇尔-露西亚的星魂双重人格间的斗争，也是裂生的星魂内心间的斗争，代表人性的希薇尔和代表星魂原始掠夺本性的露西亚的斗争。
不过这样的想法让我很担心，因为这么做会很大程度上对原作进行改动，而且某些设定存在很多漏洞，所以这还仅算是一个提议吧！<title>6</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
