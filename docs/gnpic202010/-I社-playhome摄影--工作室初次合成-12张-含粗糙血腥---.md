# 【I社 playhome摄影】 工作室初次合成 12张 含粗糙血腥...

作者：wenjiamo

TID：26861

 

# 1

初次接触playhome闲的无事，玩了一下工作室，12张耗时大概2给小时左右吧，简单后期了一下。

总体来说还是可以的，贴图材质很高，在插件加持的情况下，工作室画质有了很好的提升，并配有8K截图插件。很多地方可以细化的。

国内的UGG论坛上有一个可对每根脚趾调节的mod插件，不过价位偏高，等对pose熟悉后可以考虑一下。

缺点也还是比较明显的。

1。在人物放大的情况下，无法通过插件对人物进行打光  。

2。pose比较少，而且摆pose非常费劲，需要每一给骨骼进行移动，非常的不容易。（即使加上论坛的pose分享也是很少，或者说是不适合）

另，论坛里有没有接触过这个的，或者对调色以及角度有见解的，希望给出一些建议，我闲下来还是比较想往细致一些做的。
