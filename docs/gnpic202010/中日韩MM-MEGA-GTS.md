# 4

三张图都有更新，第一张修缮了一下细节，后面两张修改了下GTS的颜色对比，这样跟背景更加协调
一般来讲，都是要在GTS图层上加一层跟背景相同颜色（比如蓝天的兰色或城市的灰色）的颜色叠加，然后把透明度缩减，这样GTS跟背景就能更好的融合，而不会显得GTS太突兀，跟背景差太多。
我原来把透明度调成20%，可能多了点，现在第一张是10%，第二张是5%，第三张因为原图比较亮，我设15%，这样效果会好很多。
