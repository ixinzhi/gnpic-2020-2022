# 1

首先，要对大家的热情鼓励和支持表示由衷的感谢，你们的留言给了我很大的鼓舞，对于一个第一次尝试漫画的人，太重要了(尤其是没有去线的草图画法，感觉很对不起观众呵呵)。昨晚犹豫再三把前四幅发了上来，今天中午再来看，居然有这么多人留言，所以今天用一整天的时间又画了5张，接着昨天的故事，根据某位朋友的要求，我们性感的女主角再一次变大了，依然是那句话，请大家多多指教！
   这个故事还差几幅没有完结，我是打算交代一下这个巨大的女神是从什么地方来的，呵呵，应该插在第一页的前面吧，大约明天下午可以完成。后天开始就要参加两周的全天考研复习班，应该没有时间再画了。本来我还有一个构思：前几天玩了《胧村正》(Wii上那个）对里面狐女的人设很感兴趣，考研复习回来，打算再画一个关于九尾狐变成巨大的美女蹂躏古代日本人民的故事，呵呵。明天下午也许能画一个那个故事的封面，希望大家多多支持！两周以后，I'll Be Back！
