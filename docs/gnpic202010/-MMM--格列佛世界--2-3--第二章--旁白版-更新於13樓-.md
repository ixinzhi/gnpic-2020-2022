反正我手賤就各種類都加減碰一下![](static/image/smiley/bigface/57.gif)
如果只是3D軟件裡的比較 可參考A大最近發的文章 [http://giantessnight.com/gnforum ... hread&tid=16035](http://giantessnight.com/gnforum2012/forum.php?mod=viewthread&tid=16035)

MMD 跟 3D軟件 算是兩種不同方面的吧
MMD就動漫方面的 3D就一般傾向寫實吧 (寫實? 你看一下A大的圖就知道啥是最高境界了)
兩者的配備要求其實差不多 硬要說的話就MMD比較低一點吧
如果要好看 其實基本上都要不錯的配件和顯卡
MMD就是從EFFECT上著手 3D軟件就從渲染過程上著手

如果新人想接觸的話 先想清楚是動漫/寫實方面吧
然後資源方面 目前MMD大熱 因此網上會有一堆模型,場景跟姿勢
3D軟件的話 以Poser的資源比較多 (但感覺最近入手這些資源的頻道少了很多<- 待求證)
相比之下, MMD比較容易入手資源方面吧

然後介面方面，MMD相比3D軟件會比較簡單, 因為3D軟件還會顧及設定關於材質的部份 (MMD相對的PMD editor 也可設置這部份) MMD算是相對好上手

如果是弄影片的話....兩邊我也沒碰過
