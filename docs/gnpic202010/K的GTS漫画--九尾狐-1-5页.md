# K的GTS漫画：【九尾狐】1-5页

作者：kunoichi99

TID：3521

 

# 1

今天下午把这个新连载的1-5页整理出来了，总体来说，其实对自己画的这个短篇不是很满意，在画的过程中发现了一些问题，在这里我把这些问题列举出来，期望大家能够多提建议，把这个瓶颈解决掉。

首先就是内容的创新性问题，GIGA级的画来画去，基本就是踩啊踩，踩啊踩，这就出现单调的问题了……再画这篇时，我开始就尽力的避免出现和上一个短篇雷同的场面，结果就遇到了“还应该怎么踩呢？”这个郁闷的问题。eventually同学的分析已经很透彻了，昨天晚上拜读了但大的《至黑之夜》，很受启发！让女主角赤足应该可以通过足部的不同动作丰富内容，这个比穿鞋的主角要灵活好多吧。此外，《至黑之夜》中的几个毁灭星球的场景太经典了，在这里向但大申请一下关于可否把《至黑之夜》中那几段场景漫画化？最后一页是我关于《至黑之夜》中巨大少女的人设的草稿，但其中有一个小改动是我把女主角的年龄提到了15-16之间，也就是少女身材了，主要是因为我画萝莉一直感觉不正……（可能是因为个人比较控御姐的缘故），这个草稿提交给但大审核。

还有就是关于第一页……因为想交代一个比较完整的故事，所以忍住了恶心画丑男orz效果不大好，大家就看文字也行……

最后一页 《至黑之夜》的人物设定预览
