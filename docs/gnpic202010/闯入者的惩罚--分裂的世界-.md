# 1

瑜希又在家里捉住了一个被饥饿逼疯的微型人流浪汉，这种突然闯入家中的入侵者总是会给屋子的主人带来不悦。虽说瑜希所居住的城市法律上十分维护微型人的人权，微型人的生命也是充分被法律明文保护的，但由于执法的困难，偶尔不安分的微型人如果惹恼了正常人市民的话，即使被悄悄杀掉，正常人类也不会因此受到任何制裁。

被瑜希捉住的男子其实也有苦衷，作为偷渡到模范市的微型人，没有任何身份证件的他做不了一切微型人可以做的工作，也享受不到社会提供给微型人的福利。这几天楼下的垃圾堆总是没有食物，在他的妻儿已经饿了两天后，他这才下定决心悄悄潜入到住户家里，希望能带走一些食物。

但很可惜，在他刚开始搜索的时候就被瑜希一眼看到。不管他如何解释，都无法让瑜希停下手中的动作。
