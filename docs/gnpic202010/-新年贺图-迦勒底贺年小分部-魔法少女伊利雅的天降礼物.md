# 1

大家2019新年快乐~跨年这天感觉总得拿出点什么东西来，但是又什么都拿不出来。于是把垫桌角的二手数位板（同学的淘汰品，入手后几乎没用过）重新拾出来了。高中毕业之后几乎就没怎么再动过画笔了，再加上差不多第一次画板绘。对于比例不对劲啦，没有明暗关系啦，甚至某个格子的构图看着有点眼熟了这样的问题，拜托能无视掉就无视掉吧。

新年头一天就不说什么骚话了。朴实的再次祝各位二零一九年心情愉快。
