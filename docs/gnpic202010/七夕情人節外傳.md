# 1

從前從前, 有個叫牛郎的傢伙.
因為父母早逝所以只好給他的哥嫂養;然而嫂子極討厭他硬叫他跟牛XX生一隻神牛出來..
因為以前傳說人上了牛或是牛強姦人會得到神奇的法力無邊神牛
牛郎雖然有開牛郎店跟無數個老太婆幹過但從來不敢嘗試人獸類的
所以只好出去外面的世界去想通這事

他走呀走來到一瀑布旁, 旁邊大石頭上顆著一些字:
"欲得神牛; 揮刀自宮"
他嘆到: 嗯看來改變我的一生必須就這麼做了
他猶豫了一下; 咬著牙根正要砍下去..
"慢著!" 只聽咖擦的一聲...
...刀子砍到他命跟子上應聲斷成兩節
一隻神牛出現在眼前

牛郎將他可憐悽慘的身世告訴給神牛聽, 連神牛都不禁流下牛淚...
"唉~我的身世跟你也差不多悽慘..我被一人一牛雙親拋棄後就只好四處流浪
好吧!我幫你實現願望, 就是讓你一位叫織女的神女下凡並跟你結婚吧!"
牛郎極為高興! 因為神女都是比人類巨大許多的, 且永遠保持年輕美貌
牛朗本身又是gts控 所以...

後來他們結婚了又很相愛, 不但過著快樂的生活且還生了小孩(據說是體外受X)
但是這件事被天庭娘娘知道了, 非常生氣要將他們拆散
親自來到人間把織女帶走
後來神牛幫牛郎背後加裝頂極牛皮電子光速發射器
讓牛郎以每秒300000公里速度去追織女; 然而神牛因把他的皮拿來做發射器的材料也犧牲了..![](static/image/smiley/bigface/40.gif)

眼見快追到了! 王母娘娘情急之下把他的胸罩解開, 然後..
用胸罩的扣勾子將宇宙銀河劃成兩半 界線就冒出了天何讓他們倆都不能碰到對方, 只能遙遙相望
之後織女只好獵奇地把千隻喜鵲們全織在一起, 做成喜鵲橋立於兩地之間
但因喜鵲橋保存期限為一天整, 且做喜鵲橋要一年的時間
所以織女只有等到每年農曆的7/7才能編織好血淋淋的喜鵲橋兩人才能有肌膚之親

到了2012農曆7/7...
