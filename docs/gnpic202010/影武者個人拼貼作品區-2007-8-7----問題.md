# 1

事情是這樣的,最近在4chan上看到我的舊作,讓我有點困惑---[http://orz.4chan.org/d/res/534263.html](http://orz.4chan.org/d/res/534263.html)---到底是誰貼過去的?
我不是很喜歡那些圖被貼出去,因為素材都是其它作者的,尤其某站的作品我用特多,這圈子又小,給人家看見總是不妥當
但幸好是4chan,這裡帖子洗得快,大概明後天就沒了.希望能看懂這行字的人,不要隨便轉貼我的圖片.對於未經允許就私自拿把別人的圖拿來當素材一事,我還是有自覺的.這不是可以高調的東西.

那麼,下面是一些新圖,請大家這邊看看就好,不要四處佈教吧
最後一張是巨人少女小便的圖片.第一次玩這種過激的,果然還是隱藏一下,有興趣的自己點開來看吧![](static/image/smiley/default/victory.gif):<title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
