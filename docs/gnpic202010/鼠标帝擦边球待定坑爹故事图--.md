# 1

虽然成图在但大看后觉得下面弄城市图比较好，但是那样工作量会很大的……而偶又是一个十分懒的人，再加上鼠标什么的弄出来就不容易，仅仅如此就画了30个小时，最后还越看越不如意……所以最后成了这样一种擦边球……（挖坑自焚：如果还有空闲的话，我试试用光影魔术手改城市俯瞰图然后合成再涂色吧……）
嘛，反正既然决定打擦边球了，那就打呗~~~
具体故事神马的还请大家先脑补吧，我可不保证还会和上一次一样写出图片故事来……
恩，擦边球也要坑爹！~~

所以这样的结局真的大丈夫？！

ps：偷笑……实际上很早以前看到黑白纹地板……偶就想画这样的国际象棋什么的了……
PS2：残念……在网上搜城市俯瞰图没找到合适的，如果直接用引序手法又觉得和画风不符……所以各种纠结……如果谁能找到好的俯瞰图什么的- -我帮你P上……（挖坑自焚去死……我是懒人。）
