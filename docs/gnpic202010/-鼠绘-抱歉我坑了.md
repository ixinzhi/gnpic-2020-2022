# [鼠绘]抱歉我坑了

作者：rce

TID：12196

 

# 1

抱歉我坑了，当初画完这幅线稿的时候兴奋的发给了一些人看然后扬言一个星期内涂完色![](static/image/smiley/bigface/63.gif) ，没想到还没开始涂就不得不把电脑借出去了，到现在电脑还不在我的手里，这个线稿也是通过各种隐藏哄骗才让那位借走电脑的人传回来的，我不知道什么时候拿到电脑所以也就无限往后推迟涂色了。![](static/image/smiley/bigface/8.gif)

嘛，因为是鼠绘，所以细节方面肯定有不好的地方，更何况还没有做应涂色要求的细节改动。![](static/image/smiley/bigface/66.gif)

总之因为拿回来了所以就先传上来了，但愿有一天能涂完色……

ps:看了一下该图片最后保存日期是3月18号……我坑了一个多月了么![](static/image/smiley/bigface/19.gif)
