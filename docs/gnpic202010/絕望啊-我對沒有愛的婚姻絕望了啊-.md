# 1

...題目是廢話

最近真的快煩死了.本想消失一段時間把問題解決好,但卻被我搞得越來越複雜...

不久前,我公司的區副總裁把我拉去相親,女方小我六歲(驚),美國出生的ABC,不會說中文,日文卻很溜...

重點是,她是合作關係企業的董事(沒有長)的孫女

我實在說不出"可惜妳不是二次元的角色,又沒有我愛的萌屬性"這種話...

要我先試著交往看看,懦弱的我沒有拒絕,拖了不過兩個禮拜

就被那老外(區副總裁是英國人)問了一句"差不多了吧"

差很遠啦,死禿頭~~~!她不是傲嬌,又沒呆毛,而且有三條座標軸,我感覺不到愛啊~~~~~~

絕望啦!我對不是傲嬌的大小姐絕望了啊!

明天還要帶她爸(5x歲無業敗家子一隻)來我家拜訪,哇哩咧我這小套房根本見不了人啊~~~~~~~

滿屋子的漫畫與PVC,還有一尊1/3的戰國山本五十六GK,我正在想是不是拿到車上去藏...

不過說起來...就這樣擺著給人看,也許能當場就嚇退人家...

這不是閃文!我是真的很幹啊!我從15歲開始就打定主意要獨身過一輩子了.

為什麼這種漫畫中才會發生的事情會降臨到我頭上!?看漫畫看太多的報應嗎??:'(

--

說起來,要不是我真的在擔心"拒絕=從公司滾蛋",或許我能有勇氣早一點說不...
