# 1

首先感谢主办方，让我边看边画，补完了一部优秀的动漫作品。（话说我工作是忙到没时间宅了...
之所以选择《战场女武神》艾莉西亚这个角色，是因为目前为止还没有人用她画过GTS同人图。（寺田画过她的死对头塞露贝利亚GTS），所以在此补上。
11月底完成的，因为论坛出问题拖到现在才发图，很抱歉。

这次的作品换了一种画风，更贴近动漫原画的效果，也算是一次大胆的尝试吧。
点击看大图，喜欢的话，欢迎使用右下角的评分系统，谢谢！
