# 1

嗯....好幾天沒上來了,似乎論壇這兩天有怪怪的情況出現?本來前天有機會上的,可是不知為何顯示論壇關閉,害我白浪費了一次室友不在的機會![](static/image/smiley/default/tongue.gif)

現在我人還在Los Angeles參加公司研修,6/4才會回台灣.同寢的兩人一個印度,一個德國,這幾天都把我當司機用....![](static/image/smiley/default/funk.gif):

因為只有我能在美國開車,同時又在美國住過,變成我在帶他們到處跑...奇了,我以前住的明明是東岸Princeton...

研修也是悶得可以,無聊.平常團體行動,私人時間又被這對黑白無常(?)給纏住,根本沒辦法上來嘛...還好今天那兩個被隔壁寢的哥倫比亞人找去狂歡了,總算有時間泡在電腦前啦...我用不會喝酒的理由混過去好宅在宿舍....![](static/image/smiley/default/victory.gif):

OK,這次更新更直接了.主題設付費,這樣有沒有流暢些?至少只要點一次.最後面兩張長門與C.C.是上次黑白無常不在時,我用筆電作的.版上似乎有人點播的樣子~~![](static/image/smiley/default/loveliness.gif):

這邊也快半夜一點了,去睡覺啦~~撐過這段軍事集訓(?),6/2公司招待去Las Vegas徹夜狂賭,到時候鐵定要把公費慰勞的2000元給他賭到一毛不剩......更爽的是只有亞洲地區分公司的有這福利,黑白無常bye bye啦~~![](static/image/smiley/default/lol.gif)(那個印度人是在英國總公司工作的,所以沒他的份....;P)

目前我在外商金控擔任市場研究員,有點類似共同基金的後台,不過我們玩的是更投機,更大把鈔票一點的東西.:P

待遇是很好沒錯,不過我一週上班七天,半年才能休個一次長假,而且多半時間沒辦法回家睡覺,感覺在台北買的房子是拿來養蚊子的,差點要連自己床的顏色都給忘掉.:'(

好啦夠啦,私事談到這為止,以後我就不提啦.
