今作が初出演の、新しいモデルさん(Aoiさん)です。

色白でふわふわとした雰囲気で、とても魅力的なモデ

ルさんです。当サイトでは、可愛い系の女性としては

1番のルックスだと思います。

撮影時も楽しんだ様子で、いい映像になりました

最近　Aoiさんの出演した作品を数本購入しましたが、やはり今まで最も巨大女役に相応しいモデルだと思います。ひと

特にVore3とStory Base Giantess7の中に、人を沢山食べた残酷な巨大女と

彼氏を求める優しい巨大彼女を完璧に演じたことを印象深くて、多数の表情で

各シーンの雰囲気を引き立てて、まるで自分は巨大Aoiさんに支配されていると感じます。

これからのご作品はAoiさん出演しているものをお願いしたくて、個人的にはNケージの小人の模型を使用すると、

巨大女と小人の連携はもっと表現できると考えています。

前にリクエストした小人を探して、次々と口に入れたり、巨大ストローで人を吸い上げたり、人を吹き飛ばして

パワーを誇るというシーンはできればAoiさんの作品にして嬉しいです。

引き続き

よろしくお願いします。

お久しぶりです。Aoiさんの動画を複数購入していただき、ありがとうございました。

彼女は表情や雰囲気が多彩でかつルックスも素晴らしいので、私もGTSモデルとして

は最高だと思っています。特にStory Based Giantess 7は私の最も気に入っている

動画の一つで、特に最後のシーンのAoiさんの表情は素晴らしいと思います。

ストーリー要素やセリフの多い作品を、これからもAoiさんで撮影してきたいと考えてい

ます。Josephさんのリクエストも、部分的になるかもしれませんが、取り入れさせて

頂きたいと思っています。
