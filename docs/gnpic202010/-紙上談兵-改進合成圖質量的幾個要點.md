# [紙上談兵]改進合成圖質量的幾個要點

作者：killer9999

TID：3441

 

# 1

照直說,本人沒甚麼合成圖的經驗(3張,就3張而已...)

但看別人的合成圖看多了,就發現了一些普通合成圖和專業合成圖的分別

也許大家看了以後,都能成為專家吧...

合成的常識不說了...圖片質量太低不要用,縮放圖時要按比例...全說的話可累死我了

那麼就說出重點吧:

1.合成素材與周邊的清晰度要一致

這點是決定合成圖能給人多少感覺的致命點

如果這點做不好,看上去就會很怪

但如果做得好,就像真的一樣了

比如背景圖是有點模糊的,那就不可以就這樣把一個清晰的素材放上去

放上去之後還要把清晰度弄得與背景相符

對於對焦方面,也是一樣的做法.

2.邊緣要弄好

這也是另一個決定真實度的重要點

可是很多合成圖沒做這個步驟

於是素材跟背景分離了...雖然還有很少的素材選的好或剪裁的好,可以忽略這個問題

方法也很簡單...只是用模糊化工具把素材邊緣起角的地方模糊化,讓它與背景融合而已

不然會感覺怪怪的,比如話,素材有一層膜包住

(題外話,GTSFX的早期影片就有這種現象)

3.光源的位置要掌握好

這個偏差了一點看不出,但偏多了就露出破綻了

常出現在縮小男的合成圖上

解決方法:沒有,只能在選材時加倍努力

4.後期效果

GTS合成圖可不是把素材貼到背景就行的...尤其是巨大化的

有常識的人也會知道,GTS站在地面上,地面會凹陷,還有壓碎建築後的反應...

這個做得好就最好了,但就算做得一般的話也會大幅增加圖片的真實度

比如論壇裡mikeyboy兄的合成圖,就把腳印也加上去

當然,不一定要這麼講究,但至少在腳部加些煙吧

小人圖比GTS圖易弄很多...因為小的素材剪裁快,後期處理也快

但能給人感覺的總是GTS圖...因為小的素材是很難做精確的處理的

我知道小人圖在國外很受歡迎,而且成品超多

但太多人制作小人圖的後果,就是質量愈來愈低

甚至出現了「圖+小人=合成圖」的情況.

但這樣能給人感覺嗎?

別說是人家,也許自己也沒有.

那麼,到底合成圖的意義何在?

我在看合成圖時不是看完圖就算,而是會把圖徹底地研究,畢竟這樣才能給出一個好的建議.

只是看圖然後在回帖中只會讚樓主而不會批評的人,跟一個膚淺的人聽人家說甚麼都附會有何分別?
