# [Raps][汉化] 职场／天使系列组图 [22p]（足／屁屁等）

作者：coolyangzc

TID：32215



# 1

*本帖最後由 coolyangzc 於 2021-12-10 19:47 編輯*

这次带来的是Raps老师的职场/天使系列（自己给取的名字）作品的汉化，目前共有22p。Raps老师正如ta自己所言，喜欢以不止一种方式描绘waifu们的巨大屁屁（和jiojio）。更重要的是，除了直接的视觉冲击之外（有一张的屁屁是真的香），其作品更能满足“广义上”的性幻想：许多设定与背景故事都有【极其微妙】的“刺激感”与“背德感”，再加上笔下的waifu们既可爱又有趣（有时还很“可怜”），让人非常容易代入进去，去遐想能和这样的waifu们一起工作/上天堂（？）。大概这就与“犹抱琵琶半遮面”有异曲同工之妙吧。也是因为其减少了其他GTS作品中“戏谑”、“欺凌”、“残忍”的一面，增加了“可爱”、“温柔”、“淘气”的一面，因此ta的作品拥有非常广的受众基础（推上基本都在千赞以上），同时也非常适合用来“换换口味”。因此，本次我带来了Raps老师两个大系列共二十余张的汉化，做个自来水安利一下，希望大家喜欢。

值得一提的是，正如前文中所述，我觉得许多作品的标题与描述文写得既巧妙又有趣，可以有效提升作品的“风味”。因此，本次汉化我会尝试把这些描述（以尽可能合适的图像元素方式）嵌入到画面中。另外，因为图片数量众多，我会在每层楼发一组图（大家可以善用“只看该作者”功能）。含/不含描述的图片与既往汉化作品都可在[此处](https://mega.nz/folder/gM0hmKDK#o2SThYgXQPuSIItVlS0B3Q)下载。这里是Raps老师的[Twitter](https://twitter.com/YoHomeboyRaps)和[Pixiv](https://www.pixiv.net/users/24985476)，多多点赞哦！祝周末愉快~注：该系列组图系Raps老师在Twitter和Pixiv上公开发布的作品，请勿用于商业用途。
