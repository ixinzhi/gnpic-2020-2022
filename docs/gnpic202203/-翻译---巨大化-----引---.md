# 【翻译】るい巨大化イラストの引セット

作者：乔治·华盛顿

TID：32424



# 1

诸位好久不见！在此提前预祝大家新年快乐！昨天，本人终于迎来了大学的第二个寒假，因此得以抽出时间翻译这幅短小的漫画。

在各位查看鄙人拙作之前，我必须指出，我实在看不懂这幅漫画及其英文内容想表明的意思，因此只能尽量贴合原文翻译（本人4级632，扪心自问确实尽最大努力了）

尽管如此，观众姥爷们也可能对内容感到迷惑，请见谅。

这部漫画的内容大概是，某个巨大少女为了压碎自己坐着的大楼而持续用力的故事？
