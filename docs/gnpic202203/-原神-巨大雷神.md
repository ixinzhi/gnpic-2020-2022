# 【原神】巨大雷神

作者：焦冥小虫xxx

TID：31611



# 1

摸鱼了张雷神。原神里雷神的性格人设真是太戳我了，如果要是gts将是绝杀。

顺便配了个短文，文笔一般，算是随便凑的。如果有大佬能配文一首就更是好了。我就抛砖引玉了。

某一处星球的城市上空，突然阴影笼罩，随着一声惊天动地的巨响。一尊，哦不两尊超巨大的棕色巨物出现在城市的上空，随后迅速落下，直接将几万平方米的广场居民区等彻底碾在了几万吨的重物之下。惊慌失措的人们费力地抬头望向天空，发现这两座巨物竟然是一双做工精美的厚底木屐，木屐各自连接着两根直冲云霄的紫色巨柱，完全让人分辨不出来这是什么。

“到了？这就是你说的放松的地方吗？”

雷电将军看着四周坑坑洼洼的湿地。这里是旅行者那个壶里的一方洞天，自己来到这里后，周围就偶尔漂浮着一些云一样的小白雾，脚边踩着的松软灰绿色泥土遍布周围，被一滩滩极浅的水潭包围着。站在这里，简直就像是站在超大地图上一样。

“啊？哦。这样吗？”指尖上的那颗小到根本看不到的黄色小点跟自己解释着。“真没想到，这些零零散散的小斑点竟然是一座座大陆吗？我的脚，竟然这么大。倒是有些难为情呢。”

认识旅行者后，就被拉到这里。说是要让自己放松一下，看在今天是这家伙生辰的份上，便依她的愿望，来到了这片洞天。

“话说回来你怎么变得这么小了？不过，你看起来好像很开心的样子，嗯，我这个样子很美？想看我踩下去？”

好奇怪的要求。

“罢了，今天是你生辰，就听你安排便是。这里听不太清你说的话，到我耳穴里来，免得你从我手上掉下去。”

将旅行者送入右耳中后，雷电将军看着脚底下这一块脚掌大的灰色地带，根据耳中荧的介绍，这是城市，不过因为都是虚拟成像，所以不用担心。

也不知道为什么，自己看到脚下这些居住着肉眼不可见的人类城市时，不用荧要求，自己就莫名升起一丝想要踩踏的欲望。

绵延几百公里的巨足缓缓升起，带动着百万吨重的木屐从已被压平的废址再次拔地而起。这次，巨足落在了另一边完好的城市上。

零零碎碎地，在脚底踩碎了呢。

微弱的脆裂感，通过厚厚的鞋底传达到脚掌，就跟秋天踩在干枯的叶子上的感觉一样，并没什么特别。但是不知道为什么，一想到这是自己一脚踩出的景象，本不该拥有的愉悦感从心头升起。

“呵呵，有趣呢。”

内心的“影”，还从来未接触过这样的游戏。在她仔细品味刚才的微妙感时，另一只脚不自觉地擅自踩了下去。

雷神巨大的身体踩在整块大陆上，以神明的姿态傲视着周围渺小的一切，从开始到现在只是抬了抬脚，就让这大陆的绿色消失了大半。

耳穴里的荧也很开心。穿着鞋子踩了几次后，那份微妙的感觉就越来越少了。自己的身体，就潜意识地想要更多地刺激。

“脱掉鞋子踩得话，会不会更舒服些呢~”

这是这片大陆小人听到的最后一句话。
