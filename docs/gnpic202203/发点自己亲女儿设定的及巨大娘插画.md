# 1

简单介绍下角色，娜塔莉娅，大家可以叫她塔子

是一只很普通的成精狐狸，充其量是一狐妖，不能叫狐仙

目前展示的所有图片，从立绘到同人图都是找画师画的，并不是我自己的原创（虽然我本身也会画画，但我目前还没有画过巨大娘相关的创作，之后也许会有）

立绘已经得到由中华人民共和国国家版权局统一监制的登记证书，任何未经许可在商业作品（如游戏，小说等）使用娜塔莉娅形象的行为，我保有追究法律责任的权利。

非商业化的二次创作是没有问题的。

娜塔莉娅这个角色创造出来，主要就是用来GHS的，你可以通过萌娘百科和Bilibili了解到更多娜塔莉娅人物设定的相关信息

萌娘百科：[https://zh.moegirl.org.cn/%E5%A8%9C%E5%A1%94%E8%8E%89%E5%A8%85](https://zh.moegirl.org.cn/%E5%A8%9C%E5%A1%94%E8%8E%89%E5%A8%85)

*ChaosLive·S 是娜塔莉娅作为虚拟主播所在的社团名，娜塔莉娅的所有巨大娘二次创作，运营等均与ChaosLive及ChaosLive·S 无关。

入坑十来年了这还是第一在GN发帖，今后就把这里当作主活动区了，会经常发点自己的巨大娘图或小说，我觉得很多人可能已经在推上或者国内看到或者刷到我的图了，搬运过来，某种程度上也是试一试GN的发帖功能，来报个到之类的。

12月还会有一张新衣服交付，1月还有一张giga大小的插画将要进行交付

1月的giga插画可以期待一下，因为那一张插画是，娜塔莉娅的人设画师主笔去画的。
