# 1

被巨大的足心夹在中间，享受着皮肤的摩擦与淡淡的体味，望着宽大的脚底真想紧紧趴在上面磨蹭。

距上次动笔已经好几个月了，这期间弃笔的原因很多，大部分都是生活上的事情，不过最重要的是终于成功脱单了！目前她慢慢接受了我的xp，也让我享受了足x的快感。这个图是参考她的jio照画的，不过形象还是随便整的，这段时间手生了，很多地方也是糊弄一下。。。后面会继续忙活事情，画画也会很少画，patreon那边会一直暂停着的，有兴趣和时间了再看看能不能提升下我的渣技术吧！
