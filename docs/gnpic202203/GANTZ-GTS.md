# GANTZ GTS

作者：kunoichi99

TID：32799



# 1

大家好，《GANTZ GTS》是我在春节放假期间摸鱼摸出来的一个实验性的小漫画的第一话。最近整理出来了，犹豫了很久，打算还是发出来和大家分享一下。

制作的基本方法就是Photoshop素材拼贴然后在上面Paint over,再加一些风格化滤镜。

画风方面主要参考了个人比较喜欢的gantz的作者奥浩哉的风格。

想想上一次制作GTS题材的小漫画都快是十年前的事情了。。

由于使用的素材中含有大量11区当红女团前成员的肖像，

所以恳请大家请勿将本贴中的漫画图片分享到其他任何网站中

（Pictures of this manga are not allowed to be reposted to any other sites)

只是为了避免各种潜藏的肖像权等麻烦，（当然，我估计也不会有人有这个闲心hhh

本漫画仅为交流漫画制作技术之用，

如果有感到本漫画有损于肖像权持有者的形象，请及时通过留言或站内私信联系我，

我将及时删除。
