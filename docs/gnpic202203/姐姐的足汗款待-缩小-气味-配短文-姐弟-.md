# 姐姐的足汗款待（缩小，气味，配短文，姐弟）

# 1

*本帖最後由 St2019 於 2020-12-1 13:40 編輯*

这次突发奇想整一个气味控相关的活。

图不是很清晰，原图可以去我的p站账号康康。id：site2019

姐姐平时一直有一个难以说出口的烦恼：脚非常容易出汗。有时候天热的时候，在学校才待半天就要换袜子了。湿哒哒的双足沾满了足汗，给姐姐造成了不少困扰，但对我来说却是件喜闻乐见的事情。身为巨娘控的我对姐姐的双足有着不一般的兴趣。那稚嫩的大白脚看起来秀气十足，实际上却是管不住自己体液的骚气小汗足。这种强烈的反差一直吸引着我。姐姐也不是不知道我的爱好。有时候放学回到家，在清理到令她十分厌恶的足汗前，她也会记得照顾一下我的特殊癖好。当然如果我与她关系暂时闹僵的话，这难得的奖励便会泡汤。今天又是例行娱乐时间呢。湿热的天气是令人不爽的。姐姐和我好不容易回到了家，刚进门便一屁股坐了下来，习惯性地准备应对自己的烦恼。此时的我，以不到5厘米高度的形态，带着期望的眼神站在姐姐脚边。姐姐也明白了我想干什么了。虽然有点嫌弃，但也不是第一次了。黑色不透气的小皮鞋被缓缓地从劳累的双足上脱下，顿时一股浓厚的气味扑面而来。由于我的微小体型，这味道的冲击力对我格外的大。姐姐慢慢地将左脚抬起，如同一片散着热气的乌云般，带着气味移动到了我的头顶。巨大的阴影将我笼罩，与之而来的当然还有那强烈的足汗味。姐姐开始尽可能温柔地扭动脚趾，脸上则挂着略微嫌弃的表情。但我此时正在享受脚趾和袜子摩擦的沙沙声，看着头顶的尤物不断地挑动着我，让我入了迷，以至于只能呆呆地站在这散着骚气的庞大肉体下一动不动。渐渐的，那新鲜的足汗汗滴，带着浓烈的肉体代谢物，姐姐脚上的体温，和一点点袜子上的丝线，落在了我的四周，我的脚下，我的身上。可能由于这些汗滴刚刚从姐姐足底的汗腺钻出来没多久，它们的温度令人舒适，甚至如同在淋浴一般。有的较大的汗滴砸在身上，稍微有点疼，但这已经被其他令人享受的好处弥补了。此时姐姐慢慢用指尖夹起了袜子的足尖部分。随着嗦嗦的声响，那被闷了半天的大白脚终于被解放了出来。瞬间更多的汗滴直接从那潮湿温暖的皮肤和纹路里滴了出来，冲刷着我的身体，让我浑身布满了姐姐的味道，仿佛是在标记一个奴隶的所有权。看着头顶上骚动的足底，我不禁陷入了沉思。这液体和气味为何令我如此着迷？难道潜藏着什么未知的秘密吗？观赏着姐姐柔软的脚心和不时皱起的纹路，我真想继续缩小，钻进姐姐那微小的足底汗腺，好好一探究竟。
