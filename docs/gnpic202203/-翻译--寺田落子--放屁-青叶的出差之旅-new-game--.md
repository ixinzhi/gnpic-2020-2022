之后会定期挑一些如寺田，沙弥匁，DRE，Lien，蜜蜂等老师的作品（论坛里搜索不到的作品），汉化嵌字分享给大家

想着有没有自己能做的事，回报各位朋友，就开始练习翻译和嵌字，没想到挺有意思的，希望能坚持下去吧
